# ***terminal.integrated.cursorStyleInactive***

```json
{
  "terminal.integrated.cursorStyleInactive": "line"
}
```

**Descripción:** *Configura el estilo del cursor inactivo como una línea en la terminal integrada de Visual Studio Code.*
